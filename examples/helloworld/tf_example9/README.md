tf_example8 example
=====================
This example is used to demonstrate how to use mixed precision API for TensorFlow models.  

### 1. Installation
```shell
pip install -r requirements.txt
```

### 2. Download the FP32 model
```shell
wget https://storage.googleapis.com/intel-optimized-tensorflow/models/v1_6/mobilenet_v1_1.0_224_frozen.pb
```

### 3. Run Command
```shell
python test.py
```
